# How to Use

This folder contains tutorials and guides for using the various components in this repository.

## Contents

- **Video LLAVA**: Instructions for using the Video-LLAVA model for video understanding
- **Mistral v0.3**: Guide for using Mistral v0.3 model

## Getting Started

1. Navigate to the specific subfolder you want to explore
2. Read the instructions in the respective README files
3. Run the provided code examples

For more details, check the main README in the parent directory.
